# **Cube Runner 3D**

Cube Runner 3D is a **level-based 3D runner game** created in Unity. Guide your cube through challenging obstacle courses and reach the finish line to win the level!

---

## **Features**

- Designed level with increasing difficulty
- Classic runner gameplay with simple controls (move left/right)
- Score tracking
- Colorful graphics and smooth camera controls

## **Getting Started**

### **Prerequisites**
- Unity (use the same or newer version as used during project creation)
- Git (for version control)
- *(Optional)* Visual Studio Code or any C# editor

---

## **How to Play**

- Use the **arrow keys** to move your cube left or right.
- Avoid crashing into obstacles. **Reach the finish line** to complete the level!

---

## **Installation**

### **1. Clone the Repository**
git clone https://github.com/shroud07/Cube-Runner-3D

### **2. Open in Unity**
- Open **Unity Hub** and click **Add**.
- Browse to the project folder and select it.

### **3. Run the Game**
- Open the main scene.
- Hit **Play** in the Unity editor.

---

## **Project Structure**

- `Assets/` – Contains game scenes, C# scripts, models, and other resources.
- `ProjectSettings/` – Unity project configuration files.
